# HopeBridge — NGO Platform

A modern, production-ready NGO platform built with React 19+, Vite, and Tailwind CSS.

## 🚀 Features

- **Modern Tech Stack**: React 19, Vite, Tailwind CSS, Framer Motion
- **Premium Design**: Glassmorphism, Bento Layout, Luxury UI
- **Full Responsive**: Mobile-first approach with all breakpoints
- **Dark Mode**: Complete dark mode with persistence
- **Animations**: Smooth scroll animations and micro-interactions
- **Performance**: Lazy loading, code splitting, image optimization
- **Dashboard**: Full admin dashboard with analytics
- **Donation System**: Secure donation processing
- **Volunteer Management**: Volunteer registration and tracking
- **Gallery**: Pinterest-style masonry with lightbox
- **SEO Optimized**: Meta tags, Open Graph, structured data
- **Accessibility**: WCAG 2.1 compliant

## 📦 Tech Stack

- React 19+
- Vite
- Tailwind CSS
- React Router DOM v6
- Framer Motion
- React Hook Form
- Recharts
- React Helmet Async
- React Icons
